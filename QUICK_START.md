# Speed Track - Quick Start Guide

## üöÄ Getting Started

### For Users

#### 1. Connect Your Wallet
```
1. Go to https://your-domain.com/wallet
2. Click "MetaMask" 
3. Approve connection
4. Switch to BSC Testnet if prompted
```

#### 2. Register Your Account
```
1. Enter referral code (or use "ADMIN")
2. Optional: Enter leader address
3. Click "Register"
4. Confirm transaction
```

#### 3. Activate Your Account
```
1. Select activation level (0-9)
   - Level 0: 10 USDT fee, 50 USDT max investment
   - Level 1: 50 USDT fee, 250 USDT max investment
   - Level 2: 100 USDT fee, 500 USDT max investment
   - etc.
2. Approve USDT spending
3. Click "Activate"
4. Confirm transaction
```

#### 4. Complete Your Profile (Optional)
```
1. Enter your name
2. Enter your email
3. Enter country code and mobile number
4. Click "Complete Profile"
5. Confirm transaction
```

#### 5. Start Investing
```
1. Go to Dashboard
2. Click "Invest in Current Pool"
3. Enter amount (within your level limit)
4. Approve USDT
5. Confirm investment
6. Start earning!
```

### For Admins

#### 1. Access Admin Panel
```
1. Connect wallet as contract owner
2. Green shield icon appears in header
3. Click shield icon or go to /admin
4. View system overview
```

#### 2. Manage Users
```
1. Go to "Users" tab
2. Search by User ID or Address
3. View complete user information
4. Monitor user activity
```

#### 3. Monitor Pools
```
1. Go to "Pools" tab
2. Enter pool index
3. View pool details
4. Check fill status
```
